FRLife Tracker V3.1.2

Système RP pour poser un tracker GPS sur un véhicule, suivre la position avec un boîtier, scanner un véhicule et utiliser un anti-tracker. Les actions principales se font via les items ox_inventory avec l’option Utiliser. La commande /tracker reste disponible selon la configuration.

Framework : Qbox / QBCore
Inventaire : ox_inventory
Utilisation : items ox_inventory + /tracker
Version : 3.1.2 Escrow

Les images sont incluses dans le ZIP Tebex. Elles sont à copier dans ox_inventory/web/images. Le code items est fourni pour ox_inventory/data/items.lua.

Installation Tracker
1. Acheter le script sur Tebex.
2. Récupérer l’accès FiveM Asset via Cfx/Keymaster.
3. Télécharger le ZIP Tebex avec README, images et items.
4. Mettre la resource FRLifeTracker dans resources/[frlife]/.
5. Copier les images dans ox_inventory/web/images/.
6. Copier le code items dans ox_inventory/data/items.lua.
7. Ajouter dans server.cfg :

ensure ox_lib
ensure oxmysql
ensure ox_inventory
ensure FRLifeTracker

8. Redémarrer le serveur.
9. Tester avec la commande /tracker.
